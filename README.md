# Ecommerce-Sales-Dashboard(Databricks)

##  Project Overview

This project analyzes e-commerce sales data using PySpark and visualizes insights using Databricks dashboards.

##  Tools & Technologies

* Databricks
* PySpark
* SQL
* Data Visualization

##  Key Insights

* Monthly order trends analysis
* Revenue distribution by state
* Total revenue calculation

##  Steps Performed

1. Loaded dataset using PySpark
2. Cleaned and processed data
3. Created temporary views
4. Built SQL queries
5. Developed interactive dashboards

##  Dashboard Features

* Line chart for monthly orders
* Bar chart for state-wise revenue
* KPI card for total revenue

##  Conclusion

The dashboard helps understand sales patterns and regional performance, enabling better business decisions.
